# Gamer Messenger - Локальный мессенджер для геймеров

Мессенджер с голосовой/видео связью и стримингом для компании друзей (10-15 человек).

## Функциональность

- ✅ Регистрация и авторизация
- ✅ Текстовый чат в реальном времени
- ✅ Голосовая и видео связь (WebRTC)
- ✅ Стриминг экрана/игры
- ✅ Создание текстовых и голосовых каналов
- ✅ Список онлайн пользователей
- ✅ Индикаторы активности

## Технологии

- **Сервер**: Node.js, Express, Socket.io, SQLite
- **Клиент**: HTML5, CSS3, JavaScript, WebRTC API
- **База данных**: SQLite

## Установка и запуск

### Требования
- Node.js 16+
- Современный браузер с поддержкой WebRTC

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd gamer-messenger

2. Установите зависимости сервера:

cd server
npm install

3. Запустите сервер:

npm start

4. Откройте в браузере:

http://localhost:3000
